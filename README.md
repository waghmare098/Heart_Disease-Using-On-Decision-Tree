In this project, I used the Decision Tree algorithm to build a predictive model that analyzes patient health data such as age, blood pressure, cholesterol level, heart rate, and other medical indicators.
The goal is to assist healthcare professionals in early detection and risk assessment of heart disease.
Model Building using Decision Tree Classifier
Model Evaluation using Accuracy, Precision, Recall, F1-Score, and Confusion Matrix

*Using Tools*

 - Python

 - Pandas & NumPy (Data preprocessing)

 - Matplotlib & Seaborn (Data visualization)

 - Scikit-learn (Decision Tree model)

- Spyder / PyCharm
